This is work in progress, please check back later for a more complete tutorial/demo.

The purpose of this demo is to demonstrate some of the Lift module FoBo's Bootstrap v3 features such as it's built in Bootstrap transformation snippets and script generation support. The demo will also show some examples that do not use FoBo's API functions the purpose will then be to show how to work with Lift's API functions and some Bootstrap component.

The demo uses the lift-advanced-bs3 Template Application with FoBo's Bootstrap v3.x setting as it's base

If you wish to know more about FoBo and what other benefits it brings visit FoBo Home

This demo application uses the following components:

Lift
Lift is the most powerful, most secure web framework available today.
FoBo
A Generic Front-End Toolkit Module for Lift. For developing mobile scalable responsive web applications.
Bootstrap v3.x
Sleek, intuitive, and powerful mobile first front-end framework. This component is provided via the FoBo module.
Font Awesome
The iconic font designed for Bootstrap. This component is provided via the FoBo module.
google-code-prettify
Syntax highlighting of code snippets in a web page. This component is provided via the FoBo module.

Working with Lift API functions

Before we jump in to use FoBo's built in API function's we are going to demonstrate some ways to work with FoBo's Bootstrap support and Lift to do some more or less common tasks involving Lift API function. If you are familiar with Lift and have basic knowledge of Bootstrap you will be good with just a quick skim of this section.
